The Basic Language of the Web: HTML  

TOP

posted by laura jones on Monday,june 21st 2027


All modern websites and web appications are built using three fundemental technologies: HTML, CSS and javaScript. These are the languages of the web.
In this post, let's focus on HTML. We will learn what HTML is all about, and why you too should learn it.

What is HTML?

HTML stands for Hyper Text Markup Language. It's a mark up language that web developers use to structure and describe the content of a webpage(not a programming language).
HTML consists of elements that describe different types of content: parahraphs, links, headings, images, videos, etc. Web browser understand HTML and render HTML code as websites.
In HTML, each element is made up of 3 parts:

  1. The opening tag
  2. The closing tag
  3. The actual element

You can learn more at MDN Web Docs.

Why should you learn HTML?

There are countless reasons for learning the fundemental language of the web. Here are 5 of them:

  • To be able to use the fundemental web dev language
  • To hand-craft beautiful websites instead of relying on tools like worpress or Wix
  • To build web applications
  • To impress friends
